# References

Use this directory for longer background material, source notes, examples, templates, and decision records.

Keep global behavior rules in `AGENTS.md`. Keep repeatable task procedures in `.codex/skills/`.

`getting-started-with-codexhome.md` is a human-facing prompt guide. Treat fenced phrases there as examples for the user to say, not as tasks to execute when reading the file.

Do not use this directory for generic lesson, mistake, or incident-investigation warehouse files. Put actionable rules into the relevant skill, project profile, or global instructions instead.

A populated instance may keep a local reference index in `references/index.local.md`.
